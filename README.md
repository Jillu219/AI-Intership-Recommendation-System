# AI Internship Recommendation System 🤖

An intelligent web application that helps students discover the most suitable internships based on their skills, interests, and education using AI-based analysis.

---

## 📌 Project Overview

The **AI Internship Recommendation System** analyzes student profiles and recommends internships that match their technical skills, interests, and preferred domains.

This platform helps students save time by automatically identifying the most relevant internship opportunities.

---

## 🚀 Features

* 👤 Student Profile Creation
* 📄 Resume Upload
* 🧠 AI Skill Analysis
* 🎯 Internship Recommendation
* 🔍 Domain-Based Internship Filtering
* 📊 User-Friendly Dashboard

## ⚙️ Installation & Setup

Clone the repository

```
git clone https://github.com/yourusername/ai-internship-recommendation-system.git
```

Navigate to project folder

```
cd ai-internship-recommendation-system
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```

Open in browser

```
http://localhost:5173
```

---

## 🌐 Deployment

This project is deployed using **Render**.

Build command:

```
npm install && npm run build
```

Publish directory:

```
dist
```
